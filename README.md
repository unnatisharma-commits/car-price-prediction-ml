# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview

This project predicts the selling price of used cars using machine learning techniques.

## ⚙️ Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 📊 Tasks

* Data preprocessing
* Regression (Linear Regression)
* Classification (Logistic Regression)

## 📁 Dataset

CarDekho Used Car Dataset from Kaggle

## 📈 Results

* Predicted car prices using regression
* Classified cars into Low, Medium, High categories

## 📌 Platform

Implemented using Google Colab
